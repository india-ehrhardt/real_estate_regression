# Real Estate Pricing & Regression Analysis

**Tools:** Microsoft Excel, Data Analysis ToolPak, Statistical Modeling  
**Course:** Data Analytics — Metropolitan State University of Denver

---

## Overview

Built multiple linear regression models in Excel to predict residential home sale prices. The goal was to move beyond intuition-based pricing and develop a data-driven framework for evaluating property value, producing a tiered pricing strategy benchmarked against real Denver/Arvada market data.

---

## What I Did

- **Data Cleaning:** Prepared and structured raw housing data, removing inconsistencies and formatting variables for analysis
- **Simple Regression:** Started with single-variable models (e.g. above-ground square footage vs. sale price) to understand individual relationships
- **Multiple Regression:** Built a multi-variable model incorporating lot size, year built, basement square footage, and above-ground area
- **Model Evaluation:** Interpreted R² (goodness of fit) and p-values (variable significance) to assess and refine model performance
- **Pricing Strategy:** Translated model outputs into three actionable price points:
- **Suggested Price** — model-predicted value
- **Maximum Price** — upper bound based on market comparables
- **Expected Return Price** — realistic target accounting for market conditions

---

## Key Findings

- Above-ground living area was the strongest predictor of sale price
- Year built added meaningful explanatory power, reflecting buyer preference for newer construction
- The final multiple regression model outperformed single-variable models in both R² and practical usefulness
